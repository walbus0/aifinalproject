AI105 Data Analysis Experience

Select the type of user you want to be.

Data Analyst View

You will see statistical insights, such as:

  • Daily closing price and daily return
  • Monthly return distribution
  • Volatility (standard deviation) for selected month/year
  • Full statistical summary: mean, median, variance, skewness
  • Outlier detection using z-scores
  • Regression trend line for selected year
  • Correlation metrics with other indicators
  • P-values to measure significance of trends

Stock Trader View

You will see trading‑oriented metrics, such as:

  • Open, High, Low, Close data
  • Trading volume
  • 20‑day, 50‑day, and 200‑day moving averages
  • Year‑to‑date performance
  • Support and resistance levels
  • Volatility indicators (ATR)
  • Momentum indicators (RSI, MACD)
  • Trend direction (bullish/bearish)

Curious Observer View

You will see simple, beginner‑friendly summaries, such as:

  • Price on the selected date
  • Daily percentage change
  • Year summary (total increase/decrease)
  • Fun fact about that selected year
  • Simple yearly line graph placeholder
  • "If you invested $100" growth comparison
